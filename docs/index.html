# プライバシーポリシー

MonoMap（以下「本アプリ」といいます。）における、利用者情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

## 1. 収集する利用者情報及び収集方法

本アプリは、以下の利用者情報を収集します。

### (1) 端末情報

本アプリは、利用者の端末を識別する情報（端末ID等）を収集します。

### (2) 広告識別子

本アプリは、広告表示のために、広告識別子（IDFA、ADID等）を収集します。

### (3) 位置情報

本アプリは、利用者の現在地情報（GPS情報等）を収集します。

### (4) その他

本アプリは、上記以外に、利用者が本アプリを利用するにあたり、本アプリが自動的に生成する情報（アクセスログ、エラーログ等）を収集します。

## 2. 利用目的

本アプリは、収集した利用者情報を以下の目的で利用します。

*   本アプリの提供、維持、保護及び改善のため
*   本アプリに関する情報提供のため
*   本アプリに関するお問い合わせに対応するため
*   本アプリの利用状況の分析及び改善のため
*   広告の配信及び効果測定のため

## 3. 第三者提供

本アプリは、法令で認められる場合を除き、利用者の同意を得ることなく、利用者情報を第三者に提供しません。

## 4. 利用者情報の開示、訂正、利用停止等

利用者は、本アプリが保有する利用者情報について、開示、訂正、利用停止等を請求することができます。請求があった場合、本アプリは、本人確認を行った上で、合理的な期間内に対応します。

本アプリは、必要に応じて、本ポリシーを変更することがあります。変更後の本ポリシーは、本アプリに掲載した時点から効力を生じるものとします。
